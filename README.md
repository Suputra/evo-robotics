# Simulations with robots using pybullet

This project is setup with uv. For instructions on how setup uv, check out https://suputra.github.io/snippets/uv.

## Quickstart
```
uv venv 
uv pip install -r requirements.txt
chmod +x ./sim.py
chmod +x ./plot.py
```

then run:
```
./sim.py && ./plot.py
```
